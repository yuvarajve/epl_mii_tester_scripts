==============================================================================
       Communication Profile Area (0x1000 - 0x1FFF) defined by EPSG 301
==============================================================================
 This package is a template for all objects in the Communication
 Profile Area (0x1000 - 0x1FFF) defined by the POWERLINK
 Specification EPSG 301 and amendmends.
 Default values are set according to the specifications.
 
 Below listed standards are applied in their given version:
  * EPSG 301     DS   V1.1.0    Communication Profile Specification
  * EPSG 302-A   WDP  V1.0.5    High Availability
  * EPSG 302-B   DS   V1.0.0    Multiple-ASnd
  * EPSG 302-C   WDP  V0.0.3    PollResponse Chaining
